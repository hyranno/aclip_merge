
Tool to merge audio clips.

command
```
node target/main/index.js path/to/input path/to/output
```

input file
```
{
  "clips": [
    {
      "actor": "Haruka",
      "file": "Haruka/0.wav",
      "startAt": 0
    },
    {
      "actor": "Zira",
      "file": "Zira/1.wav",
      "startAt": 1000
    }
  ]
}
```
